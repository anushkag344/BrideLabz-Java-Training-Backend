# Greeting Application (Servlet 1 & Servlet 2)

A Java Servlet-based web application developed as part of the BridgeLabz Java Training Program. This project demonstrates the implementation of greeting services using Java Servlets, JSP, and the MVC architecture. It focuses on handling HTTP requests and responses while building dynamic web applications.

## Features

- Generate personalized greeting messages
- Servlet-based request and response handling
- JSP integration for dynamic web pages
- MVC Architecture
- Maven project structure
- Apache Tomcat deployment
- Clean and modular code organization

## Technologies Used

- Java
- Servlet API
- JSP
- Apache Tomcat

- CSS

## How to Run

1. Clone the repository

```bash
git clone https://github.com/anushkag344/BrideLabz-Java-Training-Backend.git
```

2. Open the project in IntelliJ IDEA or Eclipse.

3. Configure Apache Tomcat Server.

4. Build the project

```bash
mvn clean install
```

5. Deploy the application on Tomcat.

6. Access the application

```
http://localhost:8080/
```
